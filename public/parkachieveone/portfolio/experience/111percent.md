# 111% (111퍼센트)

- 기간: 2023.10 - NOW
- 링크: https://www.111percent.net
- 기술: C#, Unity, Android/iOS, Jenkins

## 주요 기여
- 핵심 모듈 및 프레임워크 개발: Table, IAP, Login, Save/Load 시스템 구축
- DevOps & Tools: 빌드 대응 및 개발자 편의 툴(Code Generator) 제공
- Firebase Integration
  - Cloud Functions + Auth를 이용한 커스텀 인증 시스템
  - Firestore DB 연동 및 Cloud Storage 리소스 업로드
- Module Development: Mobile Notification, Ranking, Addressables 리소스 관리
- Login & IAP: Google, Apple, FB, Kakao 로그인 및 서버 검증 결제 모듈
- R&D: Photon Quantum3 매치메이킹 및 세션 관리 샘플 제작
